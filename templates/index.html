Forest Fire Weather Index Prediction

Your tool for predicting fire risk based on meteorological data

About the Dataset

This application uses the Algerian Forest Fires Dataset, which contains meteorological and environmental variables collected to analyze forest fire activity in Algeria. The key features include:

  • Temperature (°C)
  • Relative Humidity (%)
  • Wind Speed (m/s)
  • Rainfall (mm)
  • Fine Fuel Moisture Code (FFMC)
  • Duff Moisture Code (DMC)
  • Initial Spread Index (ISI)
  • Class and Region codes indicating categorical information

The main target variable predicted is the Fire Weather Index (FWI), a numerical indicator that helps estimate the risk and intensity of forest fires based on fire behavior modeling.

About the Prediction Model

Using modern machine learning techniques, a Ridge Regression model was trained on this dataset after appropriate preprocessing, including removing highly correlated features and feature scaling. This model takes the nine key features above as input and predicts the Fire Weather Index, aiding in fire risk assessment.

The model has been saved along with the scaler to ensure consistent and accurate predictions on new input data.

Understanding Fire Weather Index (FWI)

The Fire Weather Index is a key metric widely used in forest fire management. It integrates weather factors like temperature, humidity, wind, and precipitation to estimate the potential severity and spread rate of forest fires. The FWI is classified into several risk levels, typically:

  • Very Low (FWI < 5.2)
  • Low (5.2 – 11.2)
  • Moderate (11.2 – 21.3)
  • High (21.3 – 38.0)
  • Very High (38.0 – 50.0)
  • Extreme (≥ 50)

This application predicts the continuous FWI value to support early fire risk evaluation and preparedness.
